definizione e importanza
defenizione del sistema operativo os
E` il gestore delle risorse del computer sia hardware sia software del computer.
funzionamento del sistema operativo os
  • Realizzare la piattaforma di esecuzione dei delle applicazioni.
  • Gestire le risorse hardware (CPU, memoria, input, output) del computer.
  • Gestisce le risorse software (Programmi installati, dati memorizzati) del computer.
Dal prospettiva dell'utente
Il il sistema operativo serve principalmente a fornire il GUI (Graphics User Interface) e viene realizzato un specifico programma separato dal sistema operativo.
l'architittura modulare del so

processore

Il sistema operativo gestisce il processore ed i programmi in esecuzione

memoria

Il sistema operativo gestisce la memoria e dei dati dei programmi in esecuzione

file e dati

Il sistema operativo gestisce i file memorizzati sulle unità di memoria

in/output

Il sistema operativo gestisce addirittura i dispositivi di input e di output

le reti

Il sistema operativo assume il ruolo di gestire la comunicaczione e la rete

I monolitici :
Sono i sistemi operativi per i quali tutti i moduli sono eseguiti in modalità protetta
Micro-kernel :
Sono i sistemi operativi in cui tutti i moduli, tranne i gestori del processore e della memoria, sono eseguiti in modalità utente.
system call :
Trasferisce al sistema operativo la richiesta dell’operazione desiderata, e l’esecuzione del programma riprende solo dopo che il sistema operativo ha completato l’operazione richiesta.
Application program interface (API) :
E` un' interfaccia dell' S.O che assume la forma di una libreria di funzioni speciali.
Esempi di sistemi operativi
windows 10
Il sistema operativo Windows 10 è l'ultima versione di un prodotto che ha una storia ormai più che trentennale, dato che Windows 1.0, la prima edizione, venne messa sul mercato nel 1985, una data che si può considerare remota dal punto di vista della storia dell'informatica. Inoltre, Microsoft ha afferamto che non ci saranno più versione di Windows, ma verrà aggiornato ogni tanto.
Kali Linux
Kali Linux è una distribuzione Linux basata su Debian, rilasciata per la prima volta nel 2013 e correntemente gestita dalla società Offensive Security. Kali linux venne concepito quale strumento omnicomprensivo di ethical hacking per l’attuazione rapida di penetration test e analisi delle vulnerabilità sui sistemi informatici.
