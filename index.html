definizione e importanza
defenizione del sistema operativo os
E` il gestore delle risorse del computer sia hardware sia software del computer.
funzionamento del sistema operativo os
  • Realizzare la piattaforma di esecuzione dei delle applicazioni.
  • Gestire le risorse hardware (CPU, memoria, input, output) del computer.
  • Gestisce le risorse software (Programmi installati, dati memorizzati) del computer.
Dal prospettiva dell'utente
Il il sistema operativo serve principalmente a fornire il GUI (Graphics User Interface) e viene realizzato un specifico programma separato dal sistema operativo.
l'architittura modulare del so
  • Gestione del processore e dei programmi in esecuzione.
  • Gestione della memoria e dei dati dei programmi in esecuzione.
  • Gestione dei file memorizzati sulle unita di memoria (HDD, SSD).
  • Gestione dei dispositivi di input e di output.
  • Gestione della comunicaczione e della rete.
